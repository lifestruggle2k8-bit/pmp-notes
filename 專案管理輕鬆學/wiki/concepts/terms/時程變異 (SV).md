# 時程變異 (SV)

## 摘要
時程變異 (SV) 是掙值管理 (EVM) 中的一個關鍵指標，用來衡量專案在特定時間點的時程執行績效。它計算已完成工作的掙值 (EV) 與原計畫完成工作的計畫值 (PV) 之間的差異。SV > 0 表示專案進度超前，SV < 0 表示進度落後，SV = 0 表示進度符合計畫。它幫助專案經理評估時程狀態並採取糾正措施。

## 考試重點
- 計算公式：SV = 掙值 (EV) - 計畫值 (PV)。
- 意義判讀：SV > 0 為時程超前；SV < 0 為時程落後；SV = 0 為時程符合計畫。
- 屬於掙值管理 (EVM) 核心指標，用於監控專案時程績效，但其結果是以貨幣單位呈現。

## 應試提示
> 💡 務必熟記SV公式及其正負值意義，區分與SPI的應用。
