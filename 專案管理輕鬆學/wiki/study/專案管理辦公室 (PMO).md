# 專案管理辦公室 (PMO - Project Management Office)

PMO 是一個組織結構，提供項目、計劃和/或組合管理支援。PMO 的主要目標是實現項目的標準化、最佳實務和治理。

## 📖 原文內容

### PMO 的定義

**PMO（專案管理辦公室）**是一個部門或單位，負責：
- 定義和維護項目管理標準
- 提供項目管理支援和指導
- 監控項目績效
- 管理組織項目資源

### PMO 的三種類型

#### 1. 支持式 PMO (Supportive PMO)

- **角色**：作為顧問和輔導提供商
- **權力**：低，主要提供建議
- **服務**：
  - 提供最佳實務
  - 培訓和輔導
  - 模板和工具
  - 項目經理資源池
- **適用**：組織成熟度較低

#### 2. 控制式 PMO (Controlling PMO)

- **角色**：規定項目如何進行
- **權力**：中等，監督和控制
- **服務**：
  - 強制標準和流程
  - 合規檢查
  - 項目模板
  - 預算和資源管理
- **適用**：成熟度中等的組織

#### 3. 指導式 PMO (Directive PMO)

- **角色**：直接管理項目
- **權力**：高，直接控制
- **服務**：
  - 直接管理項目
  - 分配資源和預算
  - 決定項目方法論
  - 評估項目經理績效
- **適用**：高度規範的組織

### PMO 的核心職能

| 職能 | 說明 |
|------|------|
| **標準化** | 建立項目管理標準、流程和最佳實務 |
| **治理** | 確保項目符合組織治理要求 |
| **資源管理** | 分配和優化組織的項目資源 |
| **知識管理** | 維護項目知識和教訓庫 |
| **績效監控** | 監控項目和計劃的績效 |
| **培訓發展** | 提高項目經理和團隊的能力 |
| **工具和技術** | 提供項目管理工具和技術支援 |

### PMO 的價值和效益

- ✅ **標準化**：統一的項目管理方法
- ✅ **效率提升**：通過最佳實務和工具提高效率
- ✅ **風險降低**：統一的風險管理和合規
- ✅ **資源優化**：跨項目的資源分配最優化
- ✅ **知識積累**：建立組織項目管理知識庫

---

## 🎯 重點摘要

- **PMO 不是項目**：它是一個支持性組織結構，服務於其他項目。

- **三類型由弱到強**：支持式 < 控制式 < 指導式

- **核心使命是「標準化、最佳實務、治理」**：這是 PMP 考試常考的三個關鍵詞。

- **PMO 的權力取決於組織文化和成熟度**：不是所有組織都需要指導式 PMO。

- **PMO 提供杠杆作用**：通過標準化和資源共享，幫助多個項目成功。
