# 組織化結構 (Organizational Structure)

組織結構決定了權力分配、責任分配和決策流程。不同的組織結構對專案管理的影響巨大，影響 PM 的權力、資源獲取和決策速度。

## 📖 原文內容

### 主要的組織結構類型

#### 1. 職能型組織（Functional Organization）

**結構：**
```
CEO
├─ 生產部門
│  ├─ 經理
│  └─ 員工
├─ 市場部門
│  ├─ 經理
│  └─ 員工
└─ IT部門
   ├─ 經理
   └─ 員工
```

**特點：**
- 按功能分組
- 員工向職能經理報告
- PM 权力小

**優點：**
- ✓ 專業深度
- ✓ 清晰的職業路徑
- ✓ 資源專業度高

**缺點：**
- ✗ PM 權力小
- ✗ 跨部門協調難
- ✗ 專案可能不優先
- ✗ 決策慢

**PM 挑戰：**
- 必須協商獲取資源
- 難以直接管理團隊
- 跨部門溝通複雜

#### 2. 項目型組織（Projectized Organization）

**結構：**
```
CEO
├─ 專案A經理
│  ├─ 開發團隊
│  ├─ 測試團隊
│  └─ 支持團隊
└─ 專案B經理
   ├─ 開發團隊
   ├─ 測試團隊
   └─ 支持團隊
```

**特點：**
- 按專案分組
- 員工向 PM 報告
- PM 權力大

**優點：**
- ✓ PM 權力大
- ✓ 決策快
- ✓ 專案優先
- ✓ 團隊凝聚力強

**缺點：**
- ✗ 資源重複
- ✗ 專業化程度低
- ✗ 項目結束後資源浪費
- ✗ 職業發展路徑不清

**PM 優勢：**
- 完全控制團隊和資源
- 直接決策
- 高效率

#### 3. 矩陣型組織（Matrix Organization）

**結構：**
```
CEO
├─ 職能部門（生產、市場、IT）
│
└─ 專案經理
   ├─ 向職能經理獲取資源
   └─ 共享團隊成員
```

**特點：**
- 職能和專案並行
- 報告給兩個經理（PM 和 職能經理）
- 權力分享

**類型：**

| 類型 | 特點 | PM 權力 |
|------|------|--------|
| **弱矩陣** | 接近職能型 | 低 |
| **強矩陣** | 接近項目型 | 高 |
| **平衡矩陣** | 平等分享 | 中等 |

**優點：**
- ✓ 靈活性
- ✓ 資源優化
- ✓ 專業度 + 專案優先平衡

**缺點：**
- ✗ 複雜性高
- ✗ 雙重報告關係
- ✗ 衝突頻繁
- ✗ 責任不清

### 組織結構對 PM 的影響

| 面向 | 職能型 | 項目型 | 矩陣型 |
|------|--------|--------|--------|
| **PM 權力** | 低 | 高 | 中 |
| **資源控制** | 難 | 容易 | 共享 |
| **決策速度** | 慢 | 快 | 中 |
| **預算** | 部門預算 | 專案預算 | 混合 |
| **工作環境** | 穩定 | 變動 | 動態 |

---

## 🎯 重點摘要

- **組織結構決定權力分配**：影響 PM 的有效性。

- **沒有完美的結構**：每種結構都有權衡。

- **矩陣型最複雜**：需要高度的溝通和協調。

- **靈活性考慮**：現代組織經常混合結構。

- **PM 必須適應**：不能改變組織結構，但要適應。

## 📋 考試要點

**常見題型：**
- ✅ 識別題：「這是什麼組織結構？」
- ✅ 影響題：「組織結構對 PM 的影響」
- ✅ 應用題：「在特定結構下 PM 應如何行動」

**記憶技巧：**
- 🧠 **三類型記憶**：職能（權力小）→ 項目（權力大）→ 矩陣（平衡）
- 🧠 **權力順序**：職能 < 弱矩陣 < 平衡矩陣 < 強矩陣 < 項目
