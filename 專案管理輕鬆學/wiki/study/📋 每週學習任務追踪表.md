**⭐ 本週必掌握：EVM 公式速記**

```
PV = Plan Value（計畫值）
EV = Earned Value（實獲值）
AC = Actual Cost（實際成本）

SV = EV - PV（進度差異）
CV = EV - AC（成本差異）

SPI = EV / PV（進度績效指標）
CPI = EV / AC（成本績效指標）

ETC = BAC - EV / CPI（完工尚需成本）
VAC = BAC - EAC（完工變異）
TCPI = (BAC - EV) / (BAC - AC)（完工績效指標）
```

**💡 考試技巧**
- [ ] 分清楚 SV, CV（差異）vs SPI, CPI（指標）
- [ ] SV/SPI > 0 = 領先進度；< 0 = 落後進度
- [ ] CV/CPI > 0 = 成本節省；< 0 = 成本超支
