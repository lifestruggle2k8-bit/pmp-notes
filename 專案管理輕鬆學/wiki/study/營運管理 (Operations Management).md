# 營運管理 (Operations Management)

營運管理是對組織的日常業務活動進行計劃、組織、指揮和控制。與項目不同，營運是持續進行的，旨在維持組織的運作和價值。

## 📖 原文內容

### 營運的特點

| 特點 | 說明 |
|------|------|
| **持續性** | 沒有明確的終點，持續進行 |
| **重複性** | 高度重複的活動 |
| **穩定性** | 流程相對穩定，目標一致 |
| **維持** | 維持既有的產品和服務 |
| **預測性** | 結果相對可預測 |

### 專案 vs 營運

| 面向 | 專案 | 營運 |
|------|------|------|
| **時間** | 有開始和結束 | 持續進行 |
| **目標** | 創造新價值 | 維持既有價值 |
| **團隊** | 臨時組織 | 永久組織 |
| **變更** | 頻繁，通過變更管理 | 通過運營程序 |
| **結果** | 獨特的交付物 | 重複的產品/服務 |

### 專案和營運的關係

```
專案執行 → 交付新產品/服務 → 轉交給營運 → 營運維持和支持
```

專案結束後，其成果通常由營運部門接手維持。

---

## 🎯 重點摘要

- **營運是持續的**：沒有終點，是組織的日常運作。

- **專案完成後轉入營運**：項目交付的成果由營運團隊維持。

- **不同的管理方法**：項目需要靈活應對，營運需要穩定和可預測。

- **組織需要平衡**：既要通過項目創新，也要通過營運維持效率。
