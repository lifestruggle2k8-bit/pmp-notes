### ⭐ 必掌握：EVM 公式速記

```
PV (計畫值)
 ↓
EV (實獲值) ← AC (實際成本)
 ↓            ↓
SV (進度差異) ← CV (成本差異)
 ↓            ↓
SPI (進度指標) ← CPI (成本指標)
                  ↓
              EAC (完工預算)
```

### ⭐ 必掌握：合約類型決策

```
合約選擇樹：

範疇明確？
├─ YES ──→ 固定價格系列 (FP, FPIF, FPEPA)
│          └─ 買方風險低 ✅
│          └─ 賣方風險高
│
└─ NO ──→ 成本可償還系列 (CPFF, CPIF, CPAF) 或 T&M
           ├─ 完全不明確 → CPFF
           ├─ 介於中間 → T&M
           └─ 不明確但有目標 → CPIF
```

### 主要內容

| Task | 主題 | 關鍵概念 | 考試技巧 |
|------|------|---------|---------|

| **6** | 管理利害關係人期望 | 監控滿意度、維持成果 | 持續監控 |
| **7** | 確保知識轉移 | 經驗學習、知識蒐集 | OPA更新 |
| **8** | 規劃和管理溝通 | **溝通渠道計算** | n(n-1)/2 公式 |

### ⭐ 必掌握：衝突解決優先順序

```
衝突發生！

優先順序（PMP考試標準答案）：

1️⃣ 面對/解決 (Confrontation/Problem Solving)
   └─ 分析根本原因，找出雙贏解決方案 ✅✅✅

2️⃣ 妥協 (Compromise)
   └─ 雙方都有所讓步

3️⃣ 強制 (Force)
   └─ 用權力強制執行

❌ 迴避 (Avoid) - 永遠不選
❌ 撤退 (Withdraw) - 永遠不選

💡 考試技巧：衝突題90%答案都是「面對/解決」！
```
