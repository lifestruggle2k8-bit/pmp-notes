# 實獲值管理 (Earned Value Management - EVM)

實獲值管理（EVM）是專案成本和進度監控的核心方法。**這是 PMP 考試中最常出現的計算題**，必須掌握 8 個公式和其含義。

## 📖 原文內容

### EVM 的基礎概念

EVM 將三個維度整合在一起：
- **PV（計畫值）**：計畫完成到目前時間點的工作金額
- **EV（實獲值）**：實際完成工作所代表的價值
- **AC（實際成本）**：實際花費的金額

### EVM 的 8 個核心公式

#### 基礎公式（差異計算）

| 公式 | 計算 | 含義 | 判斷標準 |
|------|------|------|---------|
| **SV** (進度差異) | EV - PV | 進度進度超前或落後 | SV > 0 領先；SV < 0 落後 |
| **CV** (成本差異) | EV - AC | 成本節省或超支 | CV > 0 節省；CV < 0 超支 |

#### 績效指標（比值計算）

| 公式 | 計算 | 含義 | 判斷標準 |
|------|------|------|---------|
| **SPI** (進度績效指標) | EV ÷ PV | 進度執行效率 | SPI > 1 領先；SPI < 1 落後；SPI = 1 準時 |
| **CPI** (成本績效指標) | EV ÷ AC | 成本執行效率 | CPI > 1 節省；CPI < 1 超支；CPI = 1 按預算 |

#### 預測公式（完工預測）

| 公式 | 計算 | 含義 | 用途 |
|------|------|------|------|
| **EAC** (完工預算) | BAC ÷ CPI | 預估最終總成本 | 根據目前趨勢推估完工成本 |
| **ETC** (完工尚需) | EAC - AC | 還需花費多少錢 | 剩餘預算 |
| **VAC** (完工變異) | BAC - EAC | 預估節省或超支 | 最終預估偏差 |
| **TCPI** (完工績效指標) | (BAC - EV) ÷ (BAC - AC) | 後續必須達成的績效指標 | 如果要在預算內完成，未來的 CPI 需要達到多少 |

### EVM 計算實例

**假設情況：**
```
BAC (完工預算) = $100,000
目前進度：已完成 50%，但花了 $60,000
PV = $50,000 (計畫值：50% × $100,000)
EV = $50,000 (實獲值：實際完成的工作)
AC = $60,000 (實際成本：實際花費)
```

**計算結果：**

```
差異分析：
└─ SV = EV - PV = $50,000 - $50,000 = 0 ✅ 進度準時
└─ CV = EV - AC = $50,000 - $60,000 = -$10,000 ❌ 超支 $10,000

績效指標：
└─ SPI = EV ÷ PV = $50,000 ÷ $50,000 = 1.0 ✅ 進度準時
└─ CPI = EV ÷ AC = $50,000 ÷ $60,000 = 0.833 ❌ 超支，每花 $1 只完成 $0.833 的工作

預測分析：
└─ EAC = BAC ÷ CPI = $100,000 ÷ 0.833 = $120,000 ❌ 預估超支 $20,000
└─ ETC = EAC - AC = $120,000 - $60,000 = $60,000 (還需花費)
└─ VAC = BAC - EAC = $100,000 - $120,000 = -$20,000 (最終預估超支)
└─ TCPI = (BAC - EV) ÷ (BAC - AC) = ($100,000 - $50,000) ÷ ($100,000 - $60,000) = $50,000 ÷ $40,000 = 1.25
   ↳ 意義：後續必須達到 CPI = 1.25，才能在預算內完成（每花 $1 完成 $1.25 的工作）
```

## 🎯 重點摘要

- **EVM 的核心理念**：用一個統一的度量標準（金錢價值）同時追踪進度和成本，使兩者可比較。

- **差異 vs 績效指標**：
  - **差異（SV、CV）**：絕對值，看金額差多少
  - **績效指標（SPI、CPI）**：相對值，看效率如何（推薦用）

- **SV 和 SPI 的關係**：
  - SV = 0 ⟹ SPI = 1（進度準時）
  - SV > 0 ⟹ SPI > 1（進度領先）
  - SV < 0 ⟹ SPI < 1（進度落後）

- **CV 和 CPI 的關係**：
  - CV = 0 ⟹ CPI = 1（成本按預算）
  - CV > 0 ⟹ CPI > 1（成本節省）
  - CV < 0 ⟹ CPI < 1（成本超支）

- **EAC 是最重要的預測指標**：它告訴你如果保持目前趨勢，最終會花多少錢。

- **TCPI 的意義**：如果目前 CPI < TCPI，未來需要提高績效才能按預算完成；如果 CPI > TCPI，説明目前的績效足以按預算完成。

- **記憶口訣**：「PV 計 EV 實，AC 花；差異 SV CV，指標 SPI CPI；預測 EAC ETC VAC，完工 TCPI」

## 📋 考試要點

**計算題常見類型：**
1. **給 PV/EV/AC，求 SV/CV/SPI/CPI** → 最常考
2. **給 BAC/CPI，求 EAC** → 次常考
3. **給 TCPI，判斷後續績效需求** → 較少考但重要

**常見陷阱：**
- ❌ 搞混 SV 和 SPI（或 CV 和 CPI）
- ❌ EV 理解錯誤（EV = 完成百分比 × BAC，不是實際支付的費用）
- ❌ 公式寫反（如 CPI = AC ÷ EV 而非 EV ÷ AC）
- ❌ 忘記 TCPI 的意義（必須達到的未來績效指標）

**快速檢查法：**
- 如果 CV 和 CPI 同時出現，檢查是否一致（CV > 0 應該對應 CPI > 1）
- 如果 EAC > BAC，説明會超支，應該對應 CPI < 1

**記憶技巧：**
- 🧠 **快速計算**：
  - SV > 0 就是領先，反之落後
  - CPI > 1 就是節省，反之超支
  - EAC = BAC ÷ CPI（最重要的公式）
- 🧠 **口訣記憶**：「實獲減計算差異，實獲除實際績效；完工除績效指數，預測完成最終成本」

**練習建議：**
- 至少做 ≥20 道 EVM 計算題
- 掌握公式推導（為什麼是除不是乘）
- 能快速口算（給 20 道題限時 15 分鐘）
