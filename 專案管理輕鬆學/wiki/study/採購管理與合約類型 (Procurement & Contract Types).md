# 採購管理與合約類型 (Procurement & Contract Types)

採購管理涉及選擇供應商、簽訂合約和管理供應商關係。**合約類型的選擇決定了風險分擔，是 PMP 考試的常考題目。**

## 📖 原文內容

### 採購管理的核心過程

採購管理包括五個主要過程：
1. **規劃採購**：定義採購需求和策略
2. **執行採購**：發佈採購文件，評估建議書，選擇供應商
3. **控制採購**：監控供應商績效，管理合約變更
4. **結束採購**：驗收、結算、評估績效

### 三種主要合約類型

#### 1. 固定價格合約 (FFP - Fixed Price)

| 特點 | 詳述 |
|------|------|
| **定義** | 賣方承諾以固定價格完成工作，無論實際成本如何 |
| **買方風險** | ☑️☑️☑️ **低** - 成本可預測 |
| **賣方風險** | ☑️☑️☑️ **高** - 賣方承擔超支風險 |
| **範疇要求** | **範疇必須明確** - 無法變更範疇 |
| **適用情況** | 工作定義清楚、需求穩定、交付物明確 |
| **例子** | 建造房屋、製造標準產品 |
| **變更管理** | 任何範疇變更都會改變價格 |

**三個變型：**
- **FFP（Firm Fixed Price）**：完全固定，無調整
- **FPEPA（Fixed Price with Economic Price Adjustments）**：考慮通膨調整
- **FPIF（Fixed Price Incentive Fee）**：有績效獎勵 |

#### 2. 成本可償還合約 (CPFF - Cost Plus Fixed Fee)

| 特點 | 詳述 |
|------|------|
| **定義** | 買方支付賣方實際成本加上固定的費用 |
| **買方風險** | ☑️☑️☑️ **高** - 成本開放式，難以預測 |
| **賣方風險** | ☑️☑️☑️ **低** - 成本全額報銷 |
| **範疇要求** | **範疇不確定**也可以簽訂 |
| **適用情況** | 需求不清楚、高度創新、風險高、研發類項目 |
| **例子** | 研發專案、先期技術驗證、應急採購 |
| **激勵機制** | 無特別激勵（固定費用） |

**三個變型：**
- **CPFF（Cost Plus Fixed Fee）**：固定費用，無激勵
- **CPIF（Cost Plus Incentive Fee）**：有激勵條款
- **CPAF（Cost Plus Award Fee）**：以績效評估決定費用 |

#### 3. 時間與材料合約 (T&M - Time & Material)

| 特點 | 詳述 |
|------|------|
| **定義** | 買方按實際花費的時間和材料成本支付，通常按小時計 |
| **買方風險** | ☑️☑️ **中** - 成本可能增長，但不如 CPFF 開放 |
| **賣方風險** | ☑️☑️ **中** - 需要記錄時間和材料，但成本有保障 |
| **範疇要求** | **範疇介於明確和不明確之間** |
| **適用情況** | 難以預估工作量、需求可能變動、需要靈活調整 |
| **例子** | IT 支援、諮詢服務、維護合約 |
| **預設金額** | 通常有上限（Not-to-Exceed - NTE） |

### 合約類型選擇決策樹

```
範疇明確？
│
├─ YES → 固定價格家族 (FFP, FPEPA, FPIF)
│         買方風險低
│         ✅ 買方最安全
│
└─ NO → 成本可償還家族 (CPFF, CPIF, CPAF) 或 T&M
        買方風險高或中
        ├─ 完全不明確 → CPFF ✅
        ├─ 介於中間 → T&M ✅
        └─ 有績效目標 → CPIF ✅
```

### 各合約類型的風險分擔矩陣

| 維度 | FFP | T&M | CPFF |
|------|-----|-----|------|
| **買方成本風險** | ☑️ 低 | ☑️☑️ 中 | ☑️☑️☑️ 高 |
| **賣方成本風險** | ☑️☑️☑️ 高 | ☑️☑️ 中 | ☑️ 低 |
| **範疇確定度** | 必須明確 | 中度明確 | 可不明確 |
| **成本可預測性** | 高 | 中 | 低 |
| **激勵機制** | 可以有 | 通常無 | 可以有 |
| **買方控制力** | 通過合約 | 通過監督 | 通過參與 |

## 🎯 重點摘要

- **合約類型決定風險分擔**：選擇正確的合約類型是採購戰略的核心。

- **記住三大原則**：
  1. **範疇明確 → FFP**（買方風險低）
  2. **範疇不明確 → CPFF**（賣方風險高，買方風險低）
  3. **範疇中等 → T&M**（風險均衡）

- **考試常考的決策題**：
  - 「範疇不確定」→ 應選 CPFF 或 T&M，**不選 FFP**
  - 「買方想降低風險」→ 選 FFP
  - 「時間緊迫」→ 不代表選 T&M，要看範疇確定度

- **FFP 最嚴格**：一旦簽訂，任何範疇變更都會改變價格，需要通過正式變更流程。

- **CPFF 最靈活**：適合需求不清楚的項目，但買方需要強有力的監督機制控制成本。

- **T&M 最平衡**：介於兩者之間，適合既不能完全預測、也不想承擔全部風險的情況。

- **合約選擇與組織風險偏好**：
  - 風險規避型組織 → 傾向 FFP
  - 創新導向型組織 → 傾向 CPFF 或 T&M

## 📋 考試要點

**常見選擇題型：**
1. **情況判斷**：給定場景，選擇合適的合約類型
   - ❌ 「需求不清楚」選 FFP → 應選 CPFF
   - ✅ 「需求清楚、廠商競爭激烈」選 FFP

2. **風險分析**：問誰承擔風險
   - FFP：賣方風險高，買方風險低
   - CPFF：買方風險高，賣方風險低
   - T&M：雙方風險均衡

3. **決策題**：在給定預算和時間約束下選擇合約
   - 要考慮：成本確定性、範疇穩定性、時間限制、風險承受能力

**高頻陷阱：**
- ❌ 看到「成本超支」就怪合約 → 應該看選擇是否正確
- ❌ 看到「進度急」選 T&M → 進度和合約類型關係不大
- ❌ 以為 CPFF 最便宜 → 實際可能最貴（無成本上限）

**記憶技巧：**
- 🧠 **快速判斷**：「固定價格買方安，成本可償賣方安，時間材料都放寬」
- 🧠 **風險口訣**：「FFP 賣方苦，CPFF 買方苦，T&M 大家都不苦」
- 🧠 **選擇思路**：「先看範疇確不確定，再看買賣誰承擔風險，最後看有沒有激勵」

**練習建議：**
- 至少做 ≥15 道採購和合約類型題目
- 掌握三種合約的優缺點對比
- 能快速判斷給定情景應該用哪種合約
