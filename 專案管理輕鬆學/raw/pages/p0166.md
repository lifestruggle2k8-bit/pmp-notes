7.3 計算題公式集錦
雖然目前的 PMP 考試已經幾乎不考計算題了，但是本書還是將專案經理在實務應用時，所需要知道計算公式，進行整理。
| 名稱 | 公式 | 補充說明 |
|---|---|---|
| 溝通管道數量 | $C_n^2 = \frac{n \times (n-1)}{2}$ | n 應包括專案經理
例：如果團隊從四個人變成五個人，則溝通管道數量為：
$[5 \times (5-1)]/2 - [4 \times (4-1)]/2 = 4$ |
| 期望值 (EMV)
Expected Monetary Value | 期望值 = ∑ (機率 × 衝擊) | ∑ 代表連加，請考生注意正負號
衝擊就是對專案時程或成本的影響 |
| PERT 工期估計法
PERT Estimation | t = O+4M+P / 6
O = 樂觀值
M = 最有可能的值
P = 悲觀值 | PERT 工期估計法是屬於加權型的三
點估計法 |
| 標準差
Standard Deviation | σ = P-O / 6
O = 樂觀值
P = 悲觀值 | 標準差代表估計的精度範圍 |
| 浮時
Float/Slack | LS-ES
LS = 最晚開始時間
ES = 最早開始時間
LF-EF
LF = 最晚結束時間
EF = 最早結束時間 | 要徑上的活動，浮時 = 0 |
| 時程績效指標
Schedule Performance
Index (SPI) | SPI = EV / PV
EV = 實獲值
PV = 計畫值 | > 1 進度超前
= 1 進度符合
< 1 進度落後 |

- 敏捷管理核心概念與實務法則，包括WIP、MVP、DoD、Cynefin框架、敏捷五大關鍵要素及3-5-3法則（角色、活動、工件）。
