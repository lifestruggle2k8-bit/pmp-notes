## 🛠️ 調整同步間隔的方法

### 改為 1 分鐘同步一次

在 PowerShell（管理員）執行：

\\\powershell
Unregister-ScheduledTask -TaskName "ObsidianVaultSync" -Confirm:\False

\C:\Users\lifes\Documents\Obsidian Vault = "C:\Users\lifes\Documents\Obsidian Vault"

\MSFT_TaskExecAction = New-ScheduledTaskAction 
    -Execute "powershell.exe" 
    -Argument "-NoProfile -WindowStyle Hidden -Command \"cd '\C:\Users\lifes\Documents\Obsidian Vault'; git add -A; \\ = git diff --cached --name-only; if (\\) { git commit -m 'auto: 自動同步筆記'; git push origin main }\""

\ = New-ScheduledTaskTrigger -AtLogOn
\ = New-ScheduledTaskTrigger -Once -At (Get-Date) -RepetitionInterval (New-TimeSpan -Minutes 1) -RepetitionDuration (New-TimeSpan -Days 365)

\MSFT_TaskSettings3 = New-ScheduledTaskSettingsSet -AllowStartIfOnBatteries -DontStopIfGoingOnBatteries -StartWhenAvailable -MultipleInstances IgnoreNew

Register-ScheduledTask -TaskName "ObsidianVaultSync" -Action \MSFT_TaskExecAction -Trigger @(\, \) -Settings \MSFT_TaskSettings3 -RunLevel Highest -Force
\\\

### 關鍵參數

- \Minutes 1\ = 每 1 分鐘
- \Minutes 5\ = 每 5 分鐘（目前設置）
- \Minutes 10\ = 每 10 分鐘
- \Minutes 30\ = 每 30 分鐘
- \Hours 1\ = 每 1 小時
